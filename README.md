# McpToolsSdkBased
This project provides some MCP tools (nothing special: search files, read file, edit file, ...) based on [`io.modelcontextprotocol.sdk:1.1.1`](https://java.sdk.modelcontextprotocol.io/v1.1.1/), `mcp-json-jackson3` and Spring Boot 4 (=> 42 dependencies).

The tools are an adaptation of the tools in [McpToolPlayground](https://github.com/srogmann/McpToolPlayground). This project serves as a template for developing MCP tools.
